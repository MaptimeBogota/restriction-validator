Validador de restricciones de OSM
Acerca de Github
Has zoom para cargar los datos

Muestra las restricciones de giro de OpenStreetMap y las analiza para identificar restricciones incorrectas o innecesarias . El API de Overpass se usa para recuperar los datos.

  • Los caminos verdes representan restricciones de tipo "only_".
  • Los caminos rojos representan restricciones de tipo "no_".
  • Los caminos blancos están conectados al nodo-vía de alguna restricción, pero este camino no hace parte de alguna restricción.

Íconos de flechas completas representa calles en un solo sentido. Felchas sobre un solo lado de las líneas verdes o rojas, represenan calles en doble via, donde la restricción solamente afecta la dirección mostrada.

Esta herramienta tiene sus limitaciones:

  • Las restricciones donde el miembro "via" es un way son ignorados.
  • Las restricciones for tipos específicos de vehículos. Por ejemplo, "restriction:hgv" se ignoran.
  • Las restricciones que contienen calles para no carros (por ejemplo, ciclocaminos) se ignoran.

Este es un fork de MorbZ/restriction-validatory y hace parte del Proyecto de ciclovías
